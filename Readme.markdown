#  Project Challenge: News homepage

Your challenge is to build out this landing page from the designs provided in the starter code.

## Your users should be able to:

- View the optimal layout for the interface depending on their device's screen size
- See hover and focus states for all interactive elements on the page
  
##  How to Get the Starter Materials

1.  **Download:** Go to the [News homnepage](https://www.frontendmentor.io/challenges/news-homepage-H6SWTa1MFl) and click **"Download starter"**.
2.  **Setup:** Extract the files and initialize a Git repository.
3.  **Reference:** Use `style-guide.md` for exact hex codes and font sizes.
